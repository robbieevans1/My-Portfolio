# My Portfolio

A personal portfolio website built with Vite, React, and TypeScript. This site highlights my background, skills, and projects as a recent Computer Science graduate.

## Live Site

[View Portfolio](https://robbieevans1.github.io/My-Portfolio/)

## About

This portfolio was created to showcase my experience, projects, and interest in software development. It includes sections for an introduction, about me, projects, and contact information.

## Built With

- React
- TypeScript
- Vite
- CSS
- GitHub Pages

## Features

- Responsive layout
- Smooth scrolling navigation
- Project cards with live links
- Clean dark theme
- Contact section

## Getting Started

To run this project locally:

```bash
git clone https://github.com/robbieevans1/My-Portfolio.git
cd My-Portfolio
npm install
npm run dev
